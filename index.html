Pertanyaan Umum

Jawaban untuk pertanyaan yang sering diajukan

Apakah UangKuY benar-benar gratis?

Ya, 100% gratis selamanya. Tidak ada biaya langganan, tidak ada pembelian dalam aplikasi.

Apakah data saya aman?

Sangat aman! Semua data disimpan lokal di perangkat Anda. Tidak ada yang dikirim ke server.

Apakah perlu koneksi internet?

Tidak. UangKuY bekerja 100% offline. Anda bisa mencatat di mana saja.

Bagaimana cara backup data?

Buka menu Pengaturan → Cadangkan Data → Ekspor ke CSV. File siap dibuka di Excel.
