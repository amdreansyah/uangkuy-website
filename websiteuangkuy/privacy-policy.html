Kebijakan Privasi

Terakhir diperbarui: 20 Februari 2024

1. Informasi yang Kami Kumpulkan

UangKuY TIDAK mengumpulkan data pribadi pengguna. Semua data yang Anda masukkan dalam aplikasi:

  • Disimpan secara lokal di perangkat Anda menggunakan database SQLite
  • Tidak pernah dikirim ke server kami
  • Tidak pernah dibagikan kepada pihak ketiga
  • Tidak pernah digunakan untuk analisis atau iklan

2. Data yang Disimpan di Perangkat

Aplikasi menyimpan data berikut di perangkat Anda:

  • Data Transaksi: ID, tipe (pemasukan/pengeluaran), kategori, jumlah, tanggal, dan catatan
  • Profil Pengguna: Nama (hanya jika Anda mengisi)
  • Preferensi: Status first launch, filter terakhir yang dipilih

3. Izin Aplikasi

UangKuY tidak memerlukan izin khusus. Aplikasi hanya menggunakan penyimpanan internal perangkat untuk menyimpan data. Tidak memerlukan izin internet, kamera, kontak, lokasi, atau izin lainnya.

4. Keamanan Data

Karena data disimpan secara lokal di perangkat Anda:

  • Tidak ada risiko peretasan server
  • Tidak ada kebocoran data melalui internet
  • Anda memiliki kendali penuh atas data Anda

5. Backup Data

Fitur backup mengexport data ke file CSV yang disimpan di penyimpanan eksternal perangkat Anda. File ini hanya bisa diakses oleh Anda dan dapat dihapus kapan saja.

6. Hak Anda

Sebagai pengguna UangKuY, Anda memiliki hak untuk mengakses, menghapus, dan mengekspor data Anda kapan saja.

7. Kontak

Jika Anda memiliki pertanyaan tentang kebijakan privasi ini, silakan hubungi kami di: privacy@uangkuy.com
