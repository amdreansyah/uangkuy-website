FAQ

Pertanyaan Umum

Jawaban untuk pertanyaan yang sering diajukan tentang UangKuY

Apakah UangKuY benar-benar gratis?

Ya, UangKuY 100% gratis selamanya. Tidak ada biaya langganan, tidak ada pembelian dalam aplikasi (in-app purchases), dan tidak ada fitur premium. Semua fitur bisa digunakan gratis.

Apakah data saya aman?

Sangat aman! Semua data disimpan secara lokal di perangkat Anda menggunakan database SQLite. Tidak ada data yang dikirim ke server kami. Aplikasi juga tidak memerlukan izin internet, sehingga tidak mungkin data Anda bocor.

Apakah perlu koneksi internet?

Tidak. UangKuY bekerja 100% offline. Anda bisa mencatat transaksi kapan saja, di mana saja, tanpa koneksi internet. Cocok untuk digunakan di daerah dengan sinyal lemah atau saat hemat kuota.

Kategori apa saja yang tersedia?

Tersedia 9 kategori: Makanan, Transportasi, Belanja, Hiburan, Kesehatan, Pendidikan, Tagihan, Gaji, dan Lainnya. Setiap kategori memiliki icon dan warna yang berbeda untuk memudahkan identifikasi.

Bisakah membackup data?

Ya, Anda bisa mengexport data ke file CSV. File ini bisa dibuka di Excel, Google Sheets, atau aplikasi spreadsheet lainnya. Format backup lengkap: ID, Tipe (Pemasukan/Pengeluaran), Kategori, Jumlah, Tanggal, dan Catatan.

Bagaimana cara menghapus transaksi?

Tap pada transaksi yang ingin dihapus, akan muncul detail transaksi. Pilih tombol "Hapus" di bagian bawah. Konfirmasi penghapusan akan muncul sebelum transaksi benar-benar terhapus dari database.
