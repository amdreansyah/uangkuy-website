Izin Aplikasi

UangKuY tidak memerlukan izin khusus:

  • Tidak perlu izin internet
  • Tidak perlu izin kamera
  • Tidak perlu izin lokasi
  • Hanya penyimpanan internal

Keamanan Data

  • Data Anda aman karena tersimpan di perangkat Anda sendiri
  • Tidak ada data yang dikirim ke server manapun
  • Bahkan kami sebagai developer tidak bisa mengakses data Anda
  • Jika aplikasi dihapus, semua data ikut terhapus

Backup Data

Fitur backup CSV memungkinkan Anda mengekspor data ke file yang bisa dibuka di Excel atau Google Sheets. File ini disimpan di penyimpanan eksternal perangkat Anda dan hanya bisa diakses oleh Anda.

Privasi Anak-anak

Aplikasi ini tidak ditujukan untuk anak-anak di bawah 13 tahun. Kami tidak mengumpulkan data dari siapapun, termasuk anak-anak.

Perubahan Kebijakan
